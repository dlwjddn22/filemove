# py 실행
1. python filemove.py
# exe 컴파일 
1. 하나의 exe파일 생성이고 콘솔창 출력안되게 컴파일
   - `pyinstaller -w -F FileMove.py `
2. log 폴더, config.ini 파일 필요

# UI Editor 실행
1. ui 확장자 파일에 마우스 오른쪽 클릭
2. Edit Qt UI File(Designer) 메뉴 선택
